Week Planner

How to use:

  • Create blocks: Click and drag to create time blocks
  • Edit text: Double-click on a block to edit its text
  • Delete blocks: Select a block and press Delete key
  • Change color: Select a color from the picker before creating blocks
  • Block constraints: Blocks snap to 30-minute intervals and cannot overlap
  • Width: Blocks can span 1-7 days (Monday to Sunday)
